# Voice Stack

Use this skill for wake word, STT, TTS, microphone, permissions, native modules, or Jarvis voice-path work.

## Procedure

1. Inspect `package.json` first to confirm installed native dependencies.
2. Inspect the relevant files under `src/services/ai/` before researching alternatives.
3. Preserve fallback behavior when a native module, permission, or API key is unavailable.
4. Distinguish clearly between implemented behavior, mocked paths, and roadmap intent.

## Return

- Current voice-path entrypoints.
- Dependency or platform constraints.
- Recommended change, risk, and validation path.
